# Серверні рішення ігрової статистики
Виконано Функціонал необхідний для:

- Реплікація гравців (переміщення, стрибки).

- При натисканні на кнопку гравець створює (Spawn Actor) перед собою Static Mesh Actor (взяти будь-хто, в т.ч. зі стандартного геометричного набору), який повинен реплицироваться іншим гравцям. Фізична симуляція не потрібно 

За використання: Switch Authority, RPC (Remote Procedure Call)
